##***Overview****
This project is a Zuma Deluxe-inspired game developed using x86 Assembly and the Irvine32 library. It recreates the core gameplay mechanics including shooting balls, matching, and scoring.

The project focuses on implementing complete game logic at a low level without relying on modern frameworks or engines.

****Features***
Real-time ball shooting system
Collision detection
Score tracking
Keyboard-based controls
Console rendering
Game loop implementation
